Здесь живут двери маршрута — служебные входы внутрь автоматизации.
Одна дверь — одна папка с `route.ts`; адрес двери: `/projects/<кат>/<слаг>/api/<дверь>`.

- `core` — обложка ядра и выжимка закона; `?select=<адрес>` отдаёт один объект.
- `work` — только объекты, ждущие работы.
- `instruction` — текст одного закона по имени.
- `patch` — единственная запись: `set` · `add`/`delete` узла · `connect`/`disconnect` ребра ·
  `append` версии в историю или кейса в набор.

Что такое дверь маршрута — `ARCHITECTURE.md`, раздел «Словарь».
