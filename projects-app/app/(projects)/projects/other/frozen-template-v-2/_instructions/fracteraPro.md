# FRACTERA PRO — not available yet

`config` IS ALWAYS `null`. Leave it that way.

Placing an automation inside a page of the public site has no interface yet: there is nothing to
switch it on with and nothing to run it through. Any other value here would describe a state that does
not exist.

- Do not set `config`. Do not invent a page, a slot or a list of roles.
- If the owner asks for Fractera Pro, do not build a substitute: write a warning saying the mode has
  no interface yet, and continue with the rest of the work.
- The field and this instruction stay in place on purpose — when the mode arrives, its law is written
  here, and nothing else in the core has to move.
